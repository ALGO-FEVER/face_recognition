# face_recognition
## Face Recognition Based Attendance System

This project is a **Face Recognition Based Attendance System** designed to automate the traditional manual attendance process using Computer Vision. It provides a touchless, efficient, and reliable way to record attendance in real-time.

### **Key Features**
* **Real-Time Detection:** Identifies and recognizes faces instantly using a webcam feed.
* **Automated Logging:** Automatically records the name and timestamp of recognized individuals into a structured format (CSV/Excel or Database).
* **High Accuracy:** Utilizes advanced deep learning libraries to ensure precise facial feature matching.
* **User-Friendly Interface:** (Optional: Add this if you used Tkinter or Flask) Simple GUI for easy interaction and management.

### **Tech Stack**
* **Language:** Python
* **Libraries:** OpenCV, Face_Recognition, NumPy, Pandas, (Aur koi jo aapne use kiya ho, jaise dlib).
